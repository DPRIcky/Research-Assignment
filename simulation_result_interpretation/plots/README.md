# Planning Performance Plots - Generated Visualizations

## Overview
This directory contains comprehensive visualizations of planning algorithm performance metrics for Section 3.1 of the research report.

## Generated Plots

### 1. **01_path_length_comparison.png**
- **Metric**: Path Length (m)
- **Description**: Compares the average path lengths generated by each planning algorithm (A*, Hybrid A*, RRT*)
- **Features**: Error bars showing standard deviation, sample counts, color-coded by planner
- **Key Insight**: Shows which planner generates the most efficient (shortest) paths

### 2. **02_runtime_comparison.png**
- **Metric**: Planning Time (seconds)
- **Description**: Compares computational efficiency of different planning algorithms
- **Features**: Runtime statistics with error bars
- **Key Insight**: Demonstrates which planner is fastest for real-time applications

### 3. **03_smoothness_metrics.png**
- **Metrics**: Mean Curvature & Mean Jerk
- **Description**: Two-panel plot showing path smoothness characteristics
  - Left panel: Mean Curvature (lower is smoother)
  - Right panel: Mean Jerk (lower is smoother)
- **Key Insight**: Indicates which planner produces the most drivable/comfortable paths

### 4. **04_success_rates.png**
- **Metrics**: Planning Success Rate & Overall Success Rate
- **Description**: Shows percentage of successful planning attempts
- **Features**: Bar charts with percentage labels
- **Key Insight**: Reliability and robustness of each planning algorithm

### 5. **05_comprehensive_summary.png**
- **Description**: 2×2 grid combining all key metrics in one figure
  - Top-left: Path Length
  - Top-right: Planning Time (Runtime)
  - Bottom-left: Smoothness (Curvature)
  - Bottom-right: Success Rate
- **Purpose**: Single comprehensive figure for report/presentation

### 6. **summary_table.csv**
- **Description**: Tabular summary of all metrics with mean ± std values
- **Columns**: 
  - Planner name
  - path_length
  - planning_time
  - mean_curvature
  - mean_jerk
  - planning_success
  - overall_success
  - min_obstacle_distance
- **Purpose**: Quantitative data for tables in the report

## Key Findings (from the data)

### Path Length
- **Hybrid A***: 57.76m (most efficient)
- **RRT***: 58.66m ± 0.62
- **A***: 59.50m

### Smoothness (Curvature)
- **RRT***: 0.147 (smoothest paths)
- **Hybrid A***: 0.210
- **A***: 0.243

### Smoothness (Jerk)
- **RRT***: 2.08 ± 2.96 (best)
- **A***: 6.64 ± 13.10
- **Hybrid A***: 7.91 ± 14.79

### Success Rate
- **All planners**: 100% success rate

## Usage

To regenerate these plots:
```bash
python planning_performance_plots.py --data-dir metrics --output-dir plots
```

To filter by estimator:
```bash
python planning_performance_plots.py --data-dir metrics --output-dir plots --estimator EKF
```

To filter by CBF state:
```bash
python planning_performance_plots.py --data-dir metrics --output-dir plots --cbf ON
```
